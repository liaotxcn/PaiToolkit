## 📂 项目结构  

- **`main.go 启动文件`**
- **`download 逻辑处理`**
- **`api 接口调用`**
- **`download_data.go 下载文件存放目录`**
- **`logs 日志存放`**
- **`middleware 路由中间件`**
- **`static 静态文件`**
- **`config 配置文件`**
